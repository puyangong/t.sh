# t.sh
 一个实用的termux脚本，用来优化你的终端体验.
 A practical termux script to optimize your terminal experience .
 
首次使用

curl -fsSL https://raw.githubusercontent.com/puyangong/t.sh/refs/heads/main/t.sh | bash

或者

wget -qO- https://raw.githubusercontent.com/puyangong/t.sh/refs/heads/main/t.sh | bash

之后直接输入 t 即可快速启动脚本.
